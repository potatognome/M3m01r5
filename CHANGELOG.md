# M3m01r5 Changelog

## [0.1.0] - 2026-04-14

### Added
- Initial release of M3m01r5 memoir journalling utility.
- Drop-in `.d` descriptor schema architecture.
- Mandatory entry fields: `ENTRY_START`, `ENTRY_TYPES`, `ENTRY`.
- Optional drop-in descriptors: `ABOUT_START`, `ABOUT_END`,
  `ABOUT_FEELINGS`, `IMAGES`, `FILES`, `AUTHOR`, `TAGS`, `LOCATION`.
- YAML-backed single-file-per-entry journal store.
- Sortable and searchable entry listing.
- Interactive CLI menu with new/list/search/view/edit/delete operations.
- Workspace and project ROOT_MODES for log and config routing.
